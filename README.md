# Keen Keeper – Friendship Analytics App

## 🚀 Project Overview

**Keen Keeper** is a modern React-based web application designed to track and analyze your communication history with friends. It helps users visualize interactions such as calls, texts, and video chats in a clean and meaningful way.

---

## 🧾 Description

Keen Keeper allows users to manage their contact history and gain insights into their communication patterns. With features like filtering, timeline views, and data visualization, users can better understand how they interact with others over time.

---

## 🛠️ Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS
* 📊 Recharts
* 🔗 React Router
* 🧠 Context API
* 💡 Lucide React Icons

---

## ✨ Key Features

1. **📈 Interaction Analytics**
   Visualize communication data (Call, Text, Video) using dynamic charts.

2. **🕒 Timeline View**
   View a chronological history of interactions with clean UI components.

3. **🔍 Filter Functionality**
   Easily filter interactions by type (Call, Text, Video) for better insights.
